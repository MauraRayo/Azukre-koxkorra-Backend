# ☕ 🍬 Azukre Koxkorra – Backend
API REST desarrollada en Spring Boot para gestionar la lógica del proyecto Azukre Koxkorra. 
Este backend proporciona los servicios necesarios para la aplicación frontend, incluyendo autenticación, gestión de datos y operaciones principales del sistema.



🚀 Tecnologías utilizadas
<ul>
  <li> Java 17 </li>
    <li> Spring Boot, Spring Web, Spring Data JPA, Spring Security </li>
    <li> MySQL Workbench </li>
    <li> Maven </li>
    <li> Hibernate </li>
    <li> Swagger (OpenAPI) </li>
</ul>
